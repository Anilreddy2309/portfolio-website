## ğŸš€ Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/anilbalireddy/portfolio.git
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build for Production

```bash
npm run build
```

This builds the app for production to the `build` folder.

## ğŸ¯ Customization

### Adding Your Information

1. **Personal Details**: Update the Hero and About sections with your information
2. **Experience**: Modify the Experience component with your work history
3. **Skills**: Update the Skills component with your technical expertise
4. **Projects**: Replace the Projects component with your actual projects
5. **Contact**: Update contact information in the Contact component

### Adding Images

1. Add your profile photo as `public/profile.jpg`
2. Add project images to `public/projects/` folder
3. Update image paths in the respective components

### Adding Resume

1. Add your resume PDF as `public/resume.pdf`
2. The download functionality will automatically work
